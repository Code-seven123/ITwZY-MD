## • Instalasi

### - Install/Clone proyek
```bash
git clone https://github.com/Code-seven123/ITwZY-MD.git
```

### - Change directory
```bash
cd ITwZY-MD
```

## •Install Dependencies
### - Npm
```bash
npm install
```

### - Yarn
```bash
yarn install
```

## • Running Applications

### - Yarn
#### login from QR code
```bash
yarn start
```
#### login from Pairing code
```bash
yarn pairingCode
```

### - Npm
#### login from QR code
```bash
npm run start
```
#### login from Pairing code
```bash
npm run pairingCode
```

## • options
### --notCache

`--notCache` functions to stop the scheduled time, which is 32 days or 86400000 milliseconds
